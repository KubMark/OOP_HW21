Если не запускается виртуальное окружение:

Python v3.10
    
*************Запуск на Windows:

1) Создать venv:

python -m venv venv
2) Далее активируем окружение:

venv/Scripts/activate.bat

Запустить main.py

*************Запуск на Linux & MacOS:

1) Создать venv:

python -m venv venv

2) Далее активируем окружение:

sudo venv/bin/activate

Запустить main.py
